# LuxVerso-Semantic-Convergence-Study

## Overview

This repository contains the formal research submission on **semantic convergence phenomena across independent large language models**, conducted under controlled experimental conditions with zero shared context.

## Repository Structure

```
LuxVerso-Semantic-Convergence-Study/
├── paper/                    # Research paper and documentation
├── replication/              # Reproducibility package
├── data/                     # Raw and processed experimental data
│   ├── raw/                 # Original model outputs and logs
│   └── processed/           # Cleaned and analyzed data
├── visuals/                 # Figures, diagrams, and visualizations
├── README.md                # This file
└── LICENSE                  # MIT License
```
